## 四、今日原型：GPS 现场取证相机「FieldStamp」

**定位**：给工程/勘测/保险/物业/执法现场人员的「拍一张就是一份可存档证据」——每张照片自动烧入 GPS 坐标、海拔、方位角、时间戳、项目/工单编号水印，全部本地归档，可批量导出 PDF 巡检报告 / CSV 台账，一次买断、零上传。

**为什么是它（今日主推理由）**：① **信号动能全队列最强**——GPS 取证相机三款（Solocator #14、MilGPS #13、Site Audit Pro #34）连续第 3 日同时上行，付费面被三日排位反复实证；② **纯本地传感器计算，天然零服务端**——相机 + GPS + 气压计（海拔）+ 磁力计（方位）+ 图上文字合成全部端侧完成，完全符合硬标准；③ **头部有明确缺口**——Solocator 转订阅后差评、Site Audit Pro 重且贵（B 端定价），「干净买断 + 离线 + 不啰嗦」的中间地带空着；④ **B 端刚需转化高**——用户是带着「工地拍照要带定位水印」的明确需求来搜的，ASO 转化远高于泛工具；⑤ 与已做的 Remcard、在做的 echo-jot **完全错位**，扩宽组合到「专业现场工具」品类。

**页面结构（4 屏）**
1. **取景/拍摄（主屏）**：全屏相机预览，底部实时叠一条**信息带**（当前 GPS 经纬度 + 海拔 + 方位罗盘 + 时间 + 当前项目名）；一枚快门键；顶部切换水印模板/项目。所拍即所见，水印在拍摄瞬间就烧进画面（非事后合成，取证可信）。
2. **相册/归档**：按「项目 / 日期 / 地点」分组的本地照片墙；每张缩略图角标带坐标；支持多选 → 批量导出。
3. **导出报告**：选一批照片 → 生成 **PDF 巡检报告**（每页一图 + 坐标/时间/备注表格 + 可选地图缩略）或 **CSV 台账**（文件名/经纬度/海拔/方位/时间/项目/备注）；导出到「文件」App 或分享。
4. **项目/设置**：项目（工单）管理、水印模板编辑（字段勾选/位置/字号/logo）、坐标格式（十进制/度分秒/UTM）、单位（米/英尺）、**全数据本地声明 + 本地备份/导出**。

**核心交互流程**
```
选/建项目 → 进相机 → 底部实时信息带显示 GPS/海拔/方位/时间/项目
→ 按快门（拍摄瞬间把信息烧入图片像素）→ 本地落库归档（EXIF 也写坐标）
→ 相册按项目/日期分组 → 多选一批 → 导出 PDF 巡检报告 / CSV 台账
→ 分享或存「文件」App（全程不联网、不上传）
```

**技术要点**
- Flutter；相机用 `camera` 插件、定位用 `geolocator`（GPS/海拔）、方位用 `flutter_compass`（磁力计），气压海拔可选 `environment_sensors`。
- 水印**拍摄瞬间合成**：取一帧 → `Canvas` 叠加文字/罗盘/logo → 存 JPEG，同时把坐标写入 EXIF（`exif`/`native_exif`），保证取证可信（非事后 P 图）。
- PDF 报告用 `pdf` 包本地生成、CSV 手写拼装；本地存储 `sqflite`/`path_provider`，照片存 App 沙盒目录。
- 地图缩略（可选）：内嵌离线静态底图或直接不画地图，避免引入地图云 SDK（保持零服务端）。
- **零上传**：所有传感器与合成本地完成；iOS 不申请网络、Android manifest 不声明 INTERNET → 「离线取证、数据留本机」是商店文案卖点，也规避 B 端数据合规顾虑。
- 风险：定位精度依赖设备 GPS（室内/弱信号需提示精度）；方位罗盘需校准提示；导出 PDF 大批量时的内存（分页流式生成）。

**商店定位**
- 分类：iOS 主 Utilities（备选 Productivity）/ Play Tools。
- 关键词：gps camera / geotag camera / field camera / timestamp camera / survey photo / 取证相机 / 定位水印相机 / 工地拍照 / 巡检 / geotag photo。
- 定价：免费可拍 + 基础水印；买断 **$6.99**（偏刚需定价，对标 Solocator 订阅疲劳）解锁：批量 PDF/CSV 导出、自定义水印模板、多项目、UTM 坐标、logo 叠加。
- 文案主张：「拍一张就是一份现场证据——坐标、时间、方位烧进照片，全部留在你手机里，永不上传」。
