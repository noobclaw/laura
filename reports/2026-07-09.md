## 四、今日原型：离线间隔重复记忆卡「记得 Remcard」

**定位**：给备考党和语言学习者的「装进口袋的间隔重复记忆卡」——自带分级优质词库、也能自建牌组，复习进度全本地、发音离线可用，一次买断、永不上传。

**为什么是它**：今日**唯一新成形且跨区双证**的选题——① 付费意愿被 **AnkiMobile 美区 #3（十年长青买断）+ 中区背单词簇 7 款进 Top100（多语言变体密集）** 双向实证，是四日来最锐的新付费信号；② 间隔重复算法（SM-2/FSRS）+ 打包词库 + 本地进度 + 系统 TTS 发音，**全是确定性本地计算，天然零服务端**，完全符合硬标准；③ 头部竞品有明确质量/体验缺口——AnkiMobile 功能强但界面陈旧、$29.99 门槛高、编辑体验极客化劝退新手，中区背单词多绑死单一考纲且审美平庸，「现代干净 + 自带优质词库 + 适中买断 + 多语言可扩」的中间地带空着；④ 与主推的相机、语音候选**完全错位**，扩宽选题组合到「学习工具」新品类，且**不触碰清理/壁纸/山寨 VPN 等垃圾赛道**。

**页面结构（4 屏）**
1. **今日复习（主屏）**：顶部一条环形进度（今日待复习 X 张 / 新学 Y 张）；主体是一张大记忆卡（正面词/问题，点击翻面看释义+例句+发音按钮）；底部四档反馈键「忘记 / 困难 / 良好 / 简单」——所见即所评，SRS 据此排下次复习时间。
2. **牌组库**：按「词库/牌组」分组（内置：四六级、考研、托福雅思、日常英语、法/西/德常用词…；自建牌组入口）；每组显示进度环 + 已掌握/学习中/未学计数；点进可预览与调参（每日新卡上限、复习上限）。
3. **卡片编辑**：新建/导入卡片（正/反面 + 例句 + 标签 + 图片/发音附件），支持 CSV/Anki .apkg 本地导入；批量编辑与标签管理。
4. **统计/设置**：复习热力图、留存曲线、连续天数（streak）；SRS 算法选择（SM-2/FSRS）、每日提醒时间（本地通知）、发音语言/语速、全数据本地声明与本地备份/导出。

**核心交互流程**
```
选牌组（内置词库/自建） → 进入今日复习队列
→ 看卡片正面 → 翻面看释义+例句 → 点发音（系统 TTS 本地朗读）
→ 按「忘记/困难/良好/简单」→ SRS 本地算下次复习间隔并落库
→ 队列清空 → 展示今日统计 + streak +1 + 排明日队列
→（可选）本地通知到点提醒复习 / 导出备份到文件
```

**技术要点**
- Flutter；SRS 用成熟开源算法（SM-2 经典、FSRS 更优），**纯本地确定性计算、可单测**，进度存本地 SQLite（`drift`/`sqflite`）。
- 发音：优先系统 TTS（iOS AVSpeechSynthesizer / Android TextToSpeech）**离线朗读**，无需联网；生僻词可随词库打包音频。
- 词库：内置分级词库随包分发（或首启一次性本地下载解压），卡片支持 CSV / Anki `.apkg` **本地导入**，不依赖任何账号或云同步。
- 备份/迁移：导出为 CSV/JSON 到「文件」App，用户自管——**不做云同步就绕开了服务端**（这正是差异卖点：进度是你的、留在你机器上）。
- **零网络权限**：全部计算本地完成，iOS 不申请、Android manifest 不声明 INTERNET → 「零上传、离线可用」本身是商店文案卖点。
- 风险：内置词库版权（用公有领域/开放许可词表，或只做算法+编辑器让用户导入自有词库起步）；`.apkg` 解析工作量（可先支持 CSV，`.apkg` 作为后续）。

**商店定位**
- 分类：Education（iOS / Play 主分类）/ 备选 Productivity。
- 关键词：spaced repetition / flashcards / vocabulary / SRS / anki alternative / 背单词 / 记忆卡 / 间隔重复 / 离线背词 / offline flashcards。
- 定价：免费用内置基础词库 + 有限自建牌组；买断 **$4.99 / ¥18** 解锁全部内置词库 + 无限牌组 + 高级统计 + CSV/apkg 导入 + FSRS 算法；**不做订阅**（针对 AnkiMobile 高价买断与订阅化学习 app 的疲劳，主打「一次买断、离线私有、进度永远是你的」）。
- 文案主张：「记得——间隔重复帮你记牢，进度永远留在你手机里，不上传任何服务器」。
