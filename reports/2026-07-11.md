## 四、今日原型：挥杆/球路轨迹相机「Swing Trace」

**定位**：给高尔夫（可扩展到棒球/网球等球类）爱好者的「录一杆就出一条发光球路轨迹」——录制或导入挥杆视频，在球飞出的起止点手动打点，App 本地把球路画成一条发光曲线叠加到慢动作视频上，一键导出视频/GIF 分享，全程离线、零上传。

**为什么是它（今日主推理由）**：① **今日信号动能全场最强**——Shot Tracer 单日 #22→#5 冲进付费榜前五，付费意愿被硬实证；② **可做成纯本地、开发量可控**——不必做实时球体识别（那才是难点），采「用户点选起止/关键点 → 贝塞尔拟合发光轨迹 → 逐帧渲染叠加」的成熟简易路线，端侧 `video_player`+`Canvas`/离屏渲染即可，零服务端；③ **与队列现有项完全错位**——运动/影像细分，扩宽组合品类；④ 属今日最活跃的「单场景本地相机/影像工具买断」族，玩法可迁移到多球类形成小矩阵。**风险与短板已明确记账**：竞品缺口是最弱维度——Shot Tracer 是成熟付费头部、口碑不差，本品需靠「更干净的买断（对标其订阅/内购疲劳）+ 多球类通用 + 更快导出」找中间地带，故入队评分仅 55，定位「跑通验证品类、不作主力」。

**页面结构（4 屏）**
1. **录制/导入（主屏）**：全屏相机预览可直接录挥杆（高帧率优先），或从相册导入已有视频；底部快门/导入切换，顶部选球类模板（高尔夫/棒球/网球）。
2. **打点标注**：时间轴逐帧拖动，用户在球飞出的**起点、若干中间点、落点**打点；App 用贝塞尔/多项式拟合出球路曲线，实时预览发光轨迹粗细/颜色/拖尾。
3. **轨迹渲染/预览**：把拟合曲线按帧「渐进显现」叠加到慢动作片段上（球飞到哪、光轨画到哪），可调慢放倍率、轨迹配色、是否显示距离/角度标注（本地几何估算，标注为「示意非测量」）。
4. **导出/设置**：导出 MP4/GIF（分辨率/帧率/水印可选）、保存到相册或「文件」；设置含球类模板管理、默认轨迹样式、**全数据本地声明（不联网、不上传）**、买断解锁项管理。

**核心交互流程**
```
录制或导入挥杆视频 → 时间轴逐帧拖动 → 在球路起点/中间/落点打点
→ App 本地拟合发光轨迹曲线 → 叠加到慢动作片段并预览
→ 调样式(颜色/粗细/慢放/标注) → 导出 MP4/GIF → 存相册或「文件」/分享
（全程端侧渲染，不联网、不上传）
```

**技术要点**
- Flutter；录制用 `camera`（尽量取高帧率）、播放/逐帧定位用 `video_player`，轨迹拟合与叠加用 `CustomPainter`/离屏 `Canvas` 逐帧绘制。
- **轨迹算法走「人工打点 + 曲线拟合」而非实时球体检测**：贝塞尔/最小二乘拟合已足够出效果，规避端侧实时 CV 的性能与准确度风险（这是把开发量压到「小-中」的关键取舍）。
- 视频导出：逐帧合成后用 `ffmpeg_kit_flutter`（本地编码，不联网）导出 MP4/GIF；大文件走分段/流式渲染防 OOM。
- 距离/发射角等标注为**本地几何示意值**（基于打点像素位移的粗估），文案明确「示意、非测量」，避免夸大功能引审核/差评。
- **零上传**：录制、拟合、渲染、导出全端侧完成；iOS 不申请网络、Android manifest 不声明 INTERNET → 「你的挥杆视频不出这台手机」作差异化文案。
- 风险：不同机型高帧率支持不一（需降级）；逐帧渲染性能（低端机限分辨率）；竞品成熟——靠买断干净 + 多球类通用突围。

**商店定位**
- 分类：iOS 主 Photo & Video（备选 Sports）/ Play Sports 或 Photography。
- 关键词：shot tracer / swing tracer / golf swing / ball flight / trajectory camera / 挥杆轨迹 / 球路轨迹 / 高尔夫慢动作 / 运动轨迹相机。
- 定价：免费可录 + 基础单色轨迹 + 带水印导出；买断 **$4.99** 解锁：多球类模板、自定义轨迹样式、去水印高清导出、GIF、批量。对标 Shot Tracer 的订阅/内购疲劳打「一次买断」。
- 文案主张：「录一杆，出一条会发光的球路——慢动作 + 轨迹，全部在你手机里完成，永不上传」。
