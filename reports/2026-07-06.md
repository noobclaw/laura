## 四、今日原型：离线语音备忘「回声笔记 EchoJot」

**定位**：隐私优先的离线语音笔记 —— 按住说、松手变文字，永不联网。

**为什么是它**：今天唯一「GitHub 技术趋势 × 商店需求 × 无服务端约束反成卖点」三线交汇的选题；meetily 一天 +1409 星证明开发者/极客圈热度正盛，而移动端同定位产品稀少（多为云订阅）。

**页面结构（4 屏）**
1. **主屏=时间线**：笔记卡片流（首行加粗当标题+两行摘要+时长徽标），顶部搜索框（全文检索），底部一颗大录音键。
2. **录音屏**：按住即录（波形动画），松手→本地转写进度条→自动落卡；支持「边说边出字」实时模式（流式解码）。
3. **笔记详情**：转写全文可编辑，音频播放条与文字**逐句对齐**（点句子跳音频），导出 Markdown/纯文本/分享。
4. **设置**：模型选择（快速/精准两档，首启后台下载）、语言（中英日自动检测）、全部数据本地声明+一键导出备份。

**核心交互流程**
```
按住录音键 → 说话(波形反馈) → 松手
→ whisper 设备端转写(3s 内出首句) → AI 本地规则取首句为标题
→ 卡片落入时间线 → 点卡片可编辑/播放/导出
```

**技术要点**
- Flutter + whisper.cpp（现成 binding：`whisper_ggml` / `flutter_whisper.cpp`），tiny/base 量化模型 40-150MB，首启下载避免包体超限。
- 逐句对齐用 whisper 的 segment 时间戳，免费得到。
- 全文搜索：SQLite FTS5。零网络权限（iOS 不申请、Android manifest 不声明 INTERNET —— 本身就是商店文案卖点）。
- 风险：低端安卓机转写慢 → 提供「仅录音、稍后转写」队列兜底。

**商店定位**
- 分类：Productivity（iOS）/ Productivity（Play）。
- 关键词：voice notes offline / 语音备忘 离线 / whisper / private voice memo / 语音转文字。
- 定价：免费 30 条笔记 + 买断解锁无限（$4.99 / ¥28）；不做订阅（与云端竞品反着来，强化「买断+离线」人设）。
- 文案主张：「你的声音不出这台手机」。
