## 四、今日原型：GPS 现场取证相机「戳记 FieldStamp」

**定位**：给建筑/房产/保险/勘验人员的「拍一张就有据可查」离线取证相机 —— 每张照片自动烧入坐标、时间、方位与项目信息，全程不联网、可批量导出成报告。

**为什么是它**：今日**唯一带明确升温动能的新赛道** —— Solocator 单日 #39→#17 暴涨、MilGPS/Site Audit Pro/Print to Size 同族聚集，付费意愿被**排位跃升实证**（非静态存在）；核心能力全是 **GPS + 罗盘 + 传感器 + 图上叠字**的确定性本地计算，**天然零服务端**；头部竞品 Solocator 正因转订阅遭差评、Site Audit Pro 重而贵，「一次买断、干净好用」的中间地带空着。与主线语音候选、昨日天光候选三线错位，构成可并行的选题组合。

**页面结构（4 屏）**
1. **取景/拍摄（主屏）**：全屏相机取景，底部实时叠一条「戳记预览」（坐标·海拔·时间·方位角·当前项目名）；左侧切换戳记模板，右侧项目/相册快选；大快门键。所见即所烧。
2. **戳记编辑**：可视化编排水印内容（字段勾选、位置九宫格、字号/底色/logo）、单位（度分秒/十进制、公制/英制）、可存多套模板对应不同工种。
3. **项目相册**：按「项目」分组的取证照片流，每张显示缩略图+坐标；支持地图视图（针脚=拍摄点）、批量选择 → **导出带 EXIF 的原图 / 一键生成 PDF 巡检报告 / CSV 坐标表**。
4. **设置**：GPS 精度与校准、罗盘手动校准、全数据本地声明、导出模板。

**核心交互流程**
```
选/建项目 → 举起相机(实时戳记预览) → 按快门
→ 本地读 GPS/海拔/罗盘/时间 → Canvas 把戳记烧进照片像素 + 同步写 EXIF
→ 落入该项目相册(带地图针) 
→ 批量选 → 导出原图 / 生成 PDF 巡检报告 / CSV 坐标 → 系统分享
```

**技术要点**
- Flutter；相机用 `camera` 插件，定位用系统 CoreLocation/FusedLocation，方位用磁力计+加速度求罗盘朝向。
- 戳记合成：取一帧 → Canvas 叠字/logo → 编码回 JPEG 并写 GPS EXIF（`exif`/`native_exif`），**烧进像素**保证转发/截图后信息不丢。
- PDF 报告：本地 `pdf` 包排版（封面项目信息 + 每页照片+坐标+时间），CSV 直接文件导出。
- **零网络权限**：全部本地完成，iOS 不申请、Android manifest 不声明 INTERNET → 本身就是「合规取证不外传」的商店卖点。
- 风险：室内 GPS 漂移（显示精度圈+允许手动微调坐标）；罗盘校准（提供 8 字校准引导+误差提示）。

**商店定位**
- 分类：Photo & Video / Utilities（iOS）、Tools / Photography（Play）。
- 关键词：gps camera / geotag photo / field documentation / timestamp camera / 现场取证 相机 / 坐标水印 / construction inspection / 巡检拍照。
- 定价：免费默认戳记 + 买断 $4.99 解锁「自定义模板 + 项目分组 + 批量 PDF/CSV 导出 + 去水印」；不做订阅（针对 Solocator 转订阅的差评，强调「一次买断、离线可用」）。
- 文案主张：「拍下即存证——坐标、时间、方位，全烧在照片里，不上传任何服务器」。
