## 四、今日原型：摄影天光规划器「金光 GoldenScout」

**定位**：给风光/婚礼/无人机摄影师的「黄金时段与天光位置」离线规划器 —— 到点、到位、到光，全程不联网。

**为什么是它**：今日**唯一新成形**且三条硬标准全中的选题——① 付费意愿被 PhotoPills 十年长青 + Lumy 双区上榜**双重验证**；② 天文历（日月位置、黄金/蓝调时段、银河窗口）是**确定性本地计算，天然零服务端**；③ 头部竞品存在明确「质量/体验缺口」（PhotoPills 强但复杂丑贵、Lumy 轻但功能薄），中间地带空着。与昨日主推 EchoJot 错位互补，可作为并行的第二发选题。

**页面结构（4 屏）**
1. **今日概览（主屏）**：顶部当前定位+日期切换；一张「天光时间轴」横条（夜→蓝调→黄金→白昼→黄金→蓝调→夜，色带可视化），标注日出/日落/黄金时段起止的精确时刻；下方卡片：太阳/月亮当前方位角+高度角、月相、银河今晚可见窗口。
2. **AR 取景**：打开相机，屏幕上叠加太阳/月亮/银河的运动轨迹弧线与「几点几分在这个方位/高度」的时间刻度点——直接对着实景看「日落会落到哪座山后」。用 GPS + 罗盘 + 陀螺仪定姿态。
3. **地图规划**：地图上放一个「针」，拖动即算该点任意日期的日月出没方位连线（光线方向可视化）；支持保存机位为收藏点。
4. **设置/收藏**：单位（度/时区）、离线地图瓦片下载、收藏机位列表、全数据本地声明。

**核心交互流程**
```
选机位(当前定位/地图拖针) + 选日期
→ 本地天文历算日出日落/黄金蓝调/日月方位高度/银河窗口(毫秒级、离线)
→ 时间轴 + 数值卡片呈现
→ 举起手机进 AR，实景叠加轨迹弧 → 决定几点到、站哪、朝哪拍
→ 收藏机位/设置到点提醒(本地通知)
```

**技术要点**
- Flutter；天文核心用成熟算法/开源实现（如 SPA 太阳位置算法、`sun_calc` / `astronomia` 思路移植），日月出没与黄金时段是标准公式，**结果确定、可离线、可单测**。
- AR 叠加：用设备 GPS + 磁力计（罗盘）+ 加速度/陀螺仪求相机指向，把天体方位角/高度角投影到取景框；不依赖 ARKit/ARCore 也能做 2.5D 叠加（降级方案，扩机型覆盖）。
- 地图：iOS 用 MapKit、Android 用系统或离线 MBTiles 瓦片；底图非核心可延后。
- **零网络权限**：全部计算本地完成，iOS 不申请、Android manifest 不声明 INTERNET → 本身就是商店文案卖点。
- 风险：AR 叠加的罗盘校准精度（提供「手动校准+误差提示」兜底）；离线地图瓦片体积（做成可选下载）。

**商店定位**
- 分类：Photo & Video（iOS）/ Photography（Play）。
- 关键词：golden hour / sun position / photo planner / 黄金时段 日出日落 / 银河拍摄 / sunset calculator / 太阳方位。
- 定价：免费查当天天光 + 买断解锁「任意日期 + AR 叠加 + 收藏机位 + 离线地图」（$4.99 / ¥18）；不做订阅（对标 PhotoPills 买断制，强调「一次买断、离线可用」）。
- 文案主张：「到点、到位、到光——不用信号也能算准每一次日落」。
