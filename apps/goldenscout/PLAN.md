# GoldenScout — 摄影天光规划器 / Photography Light Planner

> BACKLOG #6（指数 62）。3 天工厂 2026-07-21 本轮落地（队首 #5 胶片相机成败在滤镜审美、缺口 0/15、不契合自动开发流水线，按 BACKLOG 建议跳取 #6：纯天文历确定性计算、工程完全可控、缺口清晰）。

## 一句话定位

给风光 / 人像 / 城市摄影人的「拍前查光」离线工具：任选地点 + 任意日期，用**纯本地天文历算法**算出日出、日落、黄金时段、蓝调时段、太阳/月亮方位角与高度角、月相，并在一张罗盘上直观标出「太阳从哪个方向升起、几点光线最好」。全程离线、无账号、零上传，一次买断。

## 为什么是它

- **纯本地零服务端**：日月位置是确定性天文历计算（Schlyter/Meeus 端口），设备端 GPS + 磁力计取当前朝向，无需任何云端 —— 天然满足项目「无服务端」硬约束。
- **付费面跨区实证**：美区 PhotoPills #32 / Sun Seeker / myLightMeter × 中区 巧摄 / Lumy，多款两区在付费榜（BACKLOG 信号分 21/40）。
- **缺口清晰**：头部 PhotoPills 功能全但重、学习曲线陡、$10.99 信息密度高劝退轻度用户；Sun Seeker 偏老、Lumy 只做太阳。中间地带「聚焦拍前查光、干净买断、上手快」有空位。
- **工程完全可控**：确定性数学 + 可单元测试，最契合自动开发流水线（对比队首胶片相机的审美不确定性）。

## 功能清单（v1 只做核心）

1. **今日光线主屏**：当前定位地名/坐标 + 日期切换；一张日晷式**罗盘**叠加太阳/月亮当前方位、日出/日落方向射线；下方时间轴精确到分列出：晨蓝调 → 晨黄金时段 → 日出 → 正午（含太阳最高仰角）→ 日落 → 暮黄金时段 → 暮蓝调 → 天黑。
2. **太阳数据**：日出/日落时刻 + **方位角**（几度、罗盘方向）、黄金时段（-4°~+6°）晨昏起止、蓝调时段（-6°~-4°）晨昏起止、民用晨光/暮光（-6°）、正午与太阳峰值仰角、太阳当前方位角/高度角。
3. **月亮数据**：月相名称 + 照亮百分比、月升/月落、月亮当前方位角/高度角。
4. **任意日期规划（Pro）**：滑动查未来/过去任意日期的光线表。
5. **地点管理（Pro）**：一键取当前 GPS 定位；手动输入经纬度保存多个机位；在保存的机位间切换重算。
6. **实景罗盘对齐**：查看今日时，若设备有磁力计，罗盘随手机朝向旋转，可物理对准太阳升落方向。

## 页面结构

- `今日光线`（TodayScreen）：罗盘 + 时间轴卡片 + 太阳/月亮数据。免费。
- `规划`（PlannerScreen）：日期选择 + 多日光线表（Pro 解锁任意日期）。
- `地点`（LocationScreen）：当前定位 / 保存机位列表 / 手动加坐标（Pro 解锁多地点）。
- `设置`：Pro 买断入口 + 隐私 + 关于（复用壳设置页 + 工具追加项）。

## 技术要点

- **天文核心** `astro.dart`：纯 Dart，无插件依赖。Schlyter 简化历（太阳 RA/Dec + 月亮含主要摄动项 ~2′ 精度）→ 共用「赤道坐标→地平坐标」转换给出方位角/高度角。事件时刻用「按分钟采样太阳高度 + 线性插值过阈」检测（日出 -0.833°、黄金 -4°/+6°、蓝调/民用 -6°），一个高度函数派生全部时段，可精确到分且天然给出事件方位角。极昼/极夜返回空事件不崩。
- 内部一律 UTC 计算，显示时 `.toLocal()` 走设备时区（当前定位场景设备时区=目标时区，正确；远距离手动机位在 UI 注明「按设备时区显示」）。
- `geolocator` 取位（ACCESS_FINE/COARSE_LOCATION），`flutter_compass` 取朝向；均为离线设备传感器。
- 罗盘用 `CustomPainter`；地点/Pro 状态 `path_provider` 单 JSON 持久化。
- **零 INTERNET 权限**：AndroidManifest `tools:node="remove"` 剥离 INTERNET，保「计算不出机」可验证卖点。v1 不上在线地图瓦片（纯方位罗盘）以保纯离线。
- 插件多（geolocator/compass），CI Gradle 内存提到 `Xmx4096m/MaxMetaspaceSize=1024m`（母版不动，仅改本 app，见 fieldstamp 先例）。
